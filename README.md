# Portfolio Ekhsan

A personal portfolio site for Ekhsan Fitri, focused on business, procurement, analytics, and Python projects.

## Overview

This site highlights current projects, core skills, education, and internship experience in a simple one-page layout.

## Features

- Hero section with a clear professional summary
- Featured projects cards
- Skills and tools section
- Education and experience timeline
- Contact section with email and GitHub link
- Light and dark theme toggle
- Responsive layout for desktop and mobile

## Tech Stack

- HTML5
- CSS3
- JavaScript

## How to View

Open `index.html` directly in a browser, or serve the folder with any static file server.
